# Notes
Homebrew is a package manager for Mac OS
- Website: https://brew.sh/
- Docs: https://docs.brew.sh/

## Howto
### Update
```
brew update --auto-update
```

### Upgrade a package
```
brew upgrade <formula>
```

### Upgrade all packages
```
brew upgrade
```
