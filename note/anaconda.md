# Notes
Anaconda is a language and package manager

# Howto

## Updating
```
conda update conda
```

Activate environment
```
conda activate py37
```

Create environment
```
conda create --name snowflakes python=3.5
```

Show environments
```
conda info --envs
```

Install package
```
conda install beautifulsoup4
```

Update python version
```
conda update python
```
